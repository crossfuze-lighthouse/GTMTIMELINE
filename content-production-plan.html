Per-Piece Motion

Five steps.
Same every time.

Every refined draft moves through the same motion. Net-new one-pagers and the master doc start at step 3. Built in HTML, exported to PDF — same workflow that produced the renewal doc.

01
Read the draft and locate the buried bottom line. Existing drafts are written context-first. The actual answer is usually three paragraphs in. Mark it.
02
Restructure to BLUF. Answer first, context second. Lead sentence is the claim. Everything else supports it.
03
Apply the terminology sweep. Standard / Pro → Foundation. Pro Plus / Enterprise → Advanced. Enterprise Plus → Prime. Reframe Now Assist · Moveworks · AI Control Tower · Workflow Data Fabric · Virtual Agent as bundled by default, not add-ons.
04
Tighten to renewal-doc voice. Declarative. Specific. No hedging. Short sentences over long. Period at end of headlines. Spaced-caps section labels.
05
Build in HTML, export to PDF. Same workflow as the renewal doc. Brand palette and design system already locked. Ship to Lee's library for ApplePrint distribution.

What Could Go Wrong

Five risks worth
naming.

Surfaced now so Jes can pressure-test and pre-empt. None are blockers. All have a defined mitigation.

The master doc is the riskiest single piece.

12-14 pages of net-new content that has to land Jeslyn's voice perfectly and set the frame for 36 other pieces. If it misfires, everything downstream inherits the miss.
MitigationOutline + page 1 review with Jes inside 48 hours of start. Calibration before scale.

Pricing is fictional and ApplePrint is going live.

If outbound runs before real pricing lands, prospects will ask and there will be no answer. Either pricing gets confirmed before Wave 1A ships, or content is written to dodge pricing claims.
MitigationDecision needed from Jes Monday. Plan B drafted: write content with offer framing, not numbers.

Reservation Management for Workplace is unconfirmed.

Joe was waiting for confirmation on whether Reservation Mgmt is included in Foundation. If the Workplace one-pager makes claims about it, the answer has to come first.
MitigationWave 4 is late June. Plenty of time to chase Joe for the answer before Workplace one-pager is built.

"Refinement, not rewrite" is an untested assumption.

The 24 existing drafts haven't been read yet. If they're worse than "junky" — wrong angle, off-voice, missing structure — refinement becomes rewrite and Wave 1B slips.
MitigationHR Crash Course is the first refined piece. Test the assumption inside 30 minutes of starting it.

No formal review checkpoint between waves.

If Wave 1A misses Jeslyn's taste on voice, Waves 1B-4 carry the same miss. No formal gate means no automatic catch.
